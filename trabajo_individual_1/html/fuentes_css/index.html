Fuentes con CSS

1. Fuente segura del Sistema

La fuente usada para este texto es una fuente segura del sistema, es un tipo de letra (tipografía) que se encuentra preinstalado en la mayoría de los dispositivos y sistemas operativos

2. Fuente descargada mediante @font-face

La fuente usada para este texto es una fuente descargada mediante @font-face, que es es una regla de CSS que permite a los desarrolladores web utilizar tipos de letra personalizados en sus páginas web.

3. Fuente enlazada desde google fonts

La fuente usada para este texto es una fuente enlazada desde google fonts, que es un servicio por el cual puedes acceder y enlazar tipografias propeidad de Google.
