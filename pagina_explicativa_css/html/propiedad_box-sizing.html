Propiedad box-sizing

La propiedad box-sizing controla cómo se calcula el tamaño de un elemento.

Ejemplo sin box-sizing (valor por defecto: content-box)

Ancho total = width + padding + border = mucho más de 225px (que sería lo que mide el width).

Si no usamos la propiedad box-sizing: border-box; nuestro contenedor no solo medirá los 200px de ancho de width, sino que se le sumarán el a este el padding y el border.

Ejemplo con box-sizing: border-box

Ancho total = 200px (incluye padding y borde).

Al usar esta propiedad el ancho total será el del width y se reajustará para que mida 200px con el padding y el border incluido.
