Propiedad display

La propiedad display le dice al navegador cómo se debe comportar un elemento en la página: si debe ocupar toda la línea, si debe estar en línea con otros, o si será un bloque.

display: inline

Elemento inline

Caja inline, NO ocupa toda la fila, no permite modificar ni el ancho ni el alto

display: block

Elemento block
Elemento block

Caja block, SI ocupa toda la fila, permite modificar el ancho y el alto

display: inline-block

Elemento inline-block

Caja inline-block, NO ocupa toda la fila, permite modificar el ancho y el alto
